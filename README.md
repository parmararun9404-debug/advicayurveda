# 🌿 Advik Ayurveda Clinic

![Version](https://img.shields.io/badge/version-1.0.0-gold)

**Advik Ayurveda Clinic** is a modern, responsive Progressive Web App (PWA) and native Android application designed for Dr. K. D. Nakum's clinic. It provides authentic Ayurvedic treatment information, service details, and seamless WhatsApp integration for patients.

---

## ✨ Key Features

- **📱 Native Android App**: Downloadable APK for a full mobile experience.
- **🌐 PWA Support**: Install the website as an app directly from your browser.
- **🌙 Dark Mode**: Premium viewing experience with a smooth theme switcher.
- **📴 Offline Access**: Essential clinic information available even without internet.
- **🔥 Agnikarma Specialization**: Detailed sections on specialized Ayurvedic heat therapy.
- **💬 Direct Booking**: Integrated WhatsApp and Appointment booking system.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Tailwind CSS (via CDN), JavaScript.
- **Icons**: Lucide Icons.
- **Animations**: AOS Library (Animate On Scroll).
- **Mobile Wrapper**: CapacitorJS.
- **Automation**: GitHub Actions (for automatic APK builds).

## 🚀 Local Development

To run the project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/parmararun9404-debug/advicayurveda.git
   ```
2. Navigate to the folder:
   ```bash
   cd advicayurveda
   ```
3. Start a local server:
   ```bash
   # Using Python
   python -m http.server 3000
   ```
4. Open your browser and go to `http://localhost:3000`.
